Rush Java
=========

Setup
------------------

``` sh
  $ python -m venv env
  $ . ./env/bin/activate
  $ pip install --extra-index-url https://pypi.acu.epita.fr/simple -r requirements.txt
```

Build
-----

### Student subject
``` sh
  $ make all-pdf
```

### Assistant subject
``` sh
  $ make all-assistant-pdf
```

### Slides
``` sh
  $ make slides
```
